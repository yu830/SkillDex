## 修改摘要

- 将 `FilterState` 从单个 `tag` / `tool` 改为 `tags[]` / `tools[]`。
- 增加 URL query 解析与序列化工具，支持 `query`、`category`、`risk`、重复 `tags`、重复 `tools`。
- 筛选状态自动同步到 URL，刷新或复制链接后可恢复。
- Tag 和 compatible tools 筛选改为 checkbox group，保留 category / risk 单选。
- 结果数量增加 active filter count，并用 `aria-live="polite"` 通知变化。
- 空状态加入清除筛选动作和克制的恢复说明。
- 缺失外链显示为 `TBD`，不编造 repo/demo/docs 链接。
- README 增加 URL query 分享说明和数据更新注意事项。
